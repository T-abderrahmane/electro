Build a complete service marketplace platform using:

Flutter (Mobile App – Android first, scalable to iOS)

Web Admin Panel (Responsive Web App)

Backend

The platform connects clients with electricians in Algeria.

Business model:
Subscription-based only (3000 DZD/month).
No commission system.
No GPS map integration.

🎯 Core Concept

Two user roles:

Client

Electrician

Electricians must have an active subscription to:

Open request details

Send offers

Access chat

Inactive electricians:

Can see request previews

Cannot open or send offers

💳 Subscription System

Electrician subscription:

3000 DZD/month

Payment via BaridiMob transfer

Upload payment proof image

Admin manually activates subscription

Valid for 30 days

Required fields:

subscription_status (active / inactive / expired)

subscription_start_date

subscription_end_date

payment_proof_image

payment_status (pending / approved / rejected)

If expired:

Electrician loses access automatically
